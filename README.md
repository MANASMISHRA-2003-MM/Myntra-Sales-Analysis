# 🛍️ Myntra Product Analysis Dashboard

## 📌 Project Overview
This project is an interactive Excel dashboard developed to analyze Myntra product performance, revenue contribution, ratings, discounts, and brand performance. The dashboard provides meaningful business insights through KPIs, Pivot Tables, Charts, and Slicers.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Visualization

## 📊 KPIs
- Total Revenue: ₹12.81 Cr
- Total Products: 21,605
- Average Rating: 2.20/5 ⭐
- Highest Revenue Product: ₹4.86 Lakh
- Average Selling Price: ₹1,507

## 📈 Visualizations
- Category-wise Revenue
- Top 10 Brands by Revenue
- Average Rating by Category
- Product Distribution by Category
- Revenue Share by Category

## 📌 Key Insights
- Generated a total revenue of ₹12.81 Cr from 21,605 products.
- Revenue is concentrated among a few top-performing categories and brands.
- Average discount offered is around 44%.
- Categories with higher product counts contribute significantly to overall business performance.
- Product ratings indicate opportunities to improve customer satisfaction.
